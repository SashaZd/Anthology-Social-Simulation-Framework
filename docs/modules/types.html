  • Defined in types.ts:184

People Requirement Requirements on who is or must be present for an action. eg: minNumPeople:2 implies that at least 2 people must be present for this action to occur

Type declaration

  • maxNumPeople: number

    Specifies the maximum number of people that must be present for this action to occur

  • minNumPeople: number

    Specifies the minimum number of people that must be present for this action to occur

  • relationshipsAbsent: string[]

    Relationships that must not occur between the participating agents in order for action to be executed eg. [siblings] relationship for the action: "kiss_romantically"

  • relationshipsPresent: string[]

    Relationships that must occur between the participating agents in order for action to be executed eg. [teacher, student] relationships for the action: "submit_homework"

  • reqType: "people"

    Defines the type of this requirement to be a People Requirement

  • specificPeopleAbsent: Agent[]

    Array of agents that must be absent for the action to be completed

  • specificPeoplePresent: Agent[]

    Array of agents that must be present for the action to be completed eg. A cook must be present at a restaurant for food to be served to a customer.
